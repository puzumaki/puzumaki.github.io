Manual testing overview

Manual testing is performed by a human and can account for about half of the WCAG success criteria. It requires contextual understanding to determine if criteria is met. The areas of consideration are keyboard navigation, visual design, and content.

To help track accessibility testing, consider using Microsoft's free plug-in for Chrome or Edge browsers called Accessibility Insights. It allows you to go through a full assessment, saving your progress to return to later and exporting the results.

As you go through the modules of this course, each test is matched with the Accessibility Insights assessment.
