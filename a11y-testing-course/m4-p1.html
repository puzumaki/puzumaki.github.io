Screen readers overview

Screen readers are a common tool to perform manual testing for assistive technology. However, web content can be navigated with many and varied technologies such as Braille keyboards, speech to text software like Dragon Naturally, or switches and motion tracking.

According to WebAIM's 2024 screen reader survey, 76% of screen reader users are blind and about 20% have low vision or another visual impairment. Other disabilities include cognitive or learning, Deafness or Hard of Hearing, and motor. The most widely-used readers were:

1. JAWS (40%)

JAWS

JAWS is Windows only software. It can be downloaded from JAWS download link. Run the installation file and follow the instructions to install. Once the installation is complete, restart your computer before using JAWS.

2. NVDA (38%)

NVDA

NonVisual Desktop Access, or NVDA, is a Windows-only available product. Download NVDA for free.

How to use: NVDA keyboard shortcuts (Deque), NVDA quick start guide, and NVDA user guide

3. VoiceOver (10%)

VoiceOver

VoiceOver is an Apple product built-in screen reader, it is only available on Apple products.

Accessing VoiceOver

  1. Access the Apple menu (commonly represented with the apple icon)
  2. Select System Systems from the menu
  3. Open the Accessibility settings from the list, VoiceOver should be listed as an option.
  4. Choose VoiceOver and Toggle on; you will receive a prompt to go through the tutorial.

How to use: VoiceOver keyboard shortcuts (Deque) and VoiceOver user guide
