Step 2: read the WAVE report

The WAVE report has two main components: the WAVE sidebar and the original page with icons inserted.

The icons correlate to the types of content identified in the sidebar:

  • errors and contrast errors (red icons),
  • alerts (yellow icons),
  • features (green icons),
  • structure (blue icons), and
  • ARIA (purple icons).

The icons are placed on the page where they were found.

Sidebar "Details" tab

The sidebar defaults to the "Details" tab containing an overview of a page's results. At the top is a summary of what WAVE found.

Below the summary is a list of all the individual items in the same order as the summarized content.

Sidebar "Reference" tab

The documentation view is the next tab, indicated by an "i" icon. It provides a description of each criteria WAVE scans for. It can help you understand what things mean, why it matters, and how to fix issues.

Access the content in one of three ways:

  • If no icons have been selected, click the "view the icon index" link in the "Reference" tab content area.
  • From the "Details" tab, each item has an "i" icon link after the label.
  • The tooltip that pops up after clicking on an icon in the main page area.

Sidebar "Structure" tab

Structure view is the 4th tab on the sidebar. It allows you to evaluate headings and the structural hierarchy of the web content.

One of the structural features is headings. Headings are important because many assistive technology users navigate by using them. Use one heading level 1 and don't skip levels between heading level 1 through 6.

Sidebar "Contrast" tab

The "Contrast" tab, the 5th tab on the sidebar, details the page's contrast errors and provides with a tool to help you fix them.

"No styles" option

Activating the "no styles" option in the sidebar will disable all styles in the original page. This can be helpful in pinpointing where errors occur and ensuring that the reading order of the page is logical.
