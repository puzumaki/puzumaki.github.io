Accessibility Testing Course

Visual design

Objective: Ensure content remains readable and functional when altered by people with vision concerns.

Tools needed: Browser zoom, Colour Contrast Analyzer or browser Developer Tools, WebAIM Color Contrast Analyzer.

Reflow

Reflow

Can the content reflow at 320px width? Content should:

  • be functional when text is exclusively magnified by 200%,
  • be zoomable on mobile devices, and
  • not require scrolling in both directions (vertically and horizontally) unless absolutely necessary by the nature of the content.
  • Tables and maps are exempt.

How to test

Keyboard

Keyboard

  • When I set the browser window to 1280px width (use can use What is My ViewPort), and
  • I use browser zoom up to 400%
    • I see all elements on the page reflow with no content or controls becoming cut off, obscured, or requiring horizontal scrolling.
  • In Firefox, when I set Zoom Text Only (View > Zoom > Zoom Text Only), and
  • I use a browser zoom up to 200%
    • I see all text on the page is resized 200% with no content or controls becoming cut off or obscured.

Contrast

Contrast

Does normal text meet at least 4.5:1 contrast ratio including hidden content like modals? Does large text (18px or more normal or 14pt or more bold), UI components, and graphics meet 3:1?

  • UI components including graphics like icons and their states must minimally meet a 3:1 ratio: Assessment: 24.1 UI components, Assessment 24.2 State changes, and Assessment 24.3 Graphics.
  • If the UI element is inactive or disabled, it is exempt (e.g. a disabled button).

How to test

Keyboard

Keyboard

  • When I compare hex codes for two adjacent colors (e.g. foreground and background, focus indicator outline and main element or background), and
  • I use the Colour Contrast Analyzer OR Developer Tools (Inspect Element) to get the hex codes, and
  • I plug in the two hex codes into the WebAIM Contrast Checker:
    • I see the applicable WCAG AA standard is "pass".

Color

Color

Is color used as the only visual means of conveying information?

  • Meaning cannot be conveyed by color alone (it needs either an additional visual differentiation or a text-based alternative): Assessment 16.1 Color as meaning
  • Essential visual information is retained in Windows High Contrast Mode: Assessment 17.1 High contrast mode

How to test

Keyboard

Keyboard

  • When I view the content in black and white, I understand the content the same as if it were in full color.
  • When on a Windows computer, when I view the content in High Contrast Mode, I understand the content the same as if it weren't in high contrast mode.

Motion

Motion

Anything that moves, blinks, or scrolls needs to give control to the user.

  • If autoplay content is longer than 5 seconds, pause, stop, or hide controls are available: Assessment 10.2 Moving content
  • Content cannot flash more than 3 times per second: Assessment 16.4 Flashing
  • Controls should be accessible via keyboard.
