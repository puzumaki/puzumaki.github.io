Module overview: Core skills of digital accessibility

Welcome to Module 3: Core skills in digital accessibility. In this module, you will:

  • learn how you can build content to be more accessible from the start,
  • be introduced to core (universal) digital accessibility skills, and
  • experience how these changes improve experience through demos and activities.

Key points

  • No matter your role at the university, you can cultivate a more inclusive, accessible, and equitable community for everyone.
  • Start small, start now.
  • Seven core skills are universal for moving towards accessible information.
