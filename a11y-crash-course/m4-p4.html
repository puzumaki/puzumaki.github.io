Save an accessible PDF

An accessible PDF is similar to a webpage: different content is tagged to properly identify what kind of content it is, and visual content is described with text. A PDF that has not been properly converted is essentially a picture of the source document.

Pictures are not inherently accessible because computers can't read the information contained in them; a screen reader would say something like: "Page 1 of 1: blank, blank, blank."

If the original document was made with accessibility in mind, exporting it to PDF with the correct settings should retain the majority of that initial work.

Elements of accessible PDFs

  • Document properties that identify the title and language
  • All the content has a proper tag assigned
  • The order the content is read in by a machine is correct
  • All irrelevant tags or content is removed
  • Headings are set correctly
  • Images, paragraphs, lists, and tables are tagged

Links have a unique issue in a PDF. Because PDFs are intended for print, the URL of the link is often the text displayed as well (e.g. instead of the text "University of Minnesota", it might be "https://umn.edu"). If your PDF exposes links like this, alternate text can be added to the link for better digital access.

For a full tour of making a PDF accessible, please take the Digital Accessibility: Create Accessible PDFs course (it has two pre-requisite courses).

How to save a PDF

Google Docs

Google Docs has some accessibility shortcomings that cannot be resolved in Google Docs, especially with tables. The University of Minnesota provides access to a tool called GrackleDocs to help make a Google Docs file ready to make an accessible PDF version.

After making the document accessible when writing it, the next step is to run GrackleDocs. In the Google Docs interface, open Extensions > GrackleDocs > Launch. This will open a new panel as it scans the document to prepare it to make an accessible PDF.

More about GrackleDocs Accessibility Checker.

Once the issues are resolved, use the GrackleDocs "Export to PDF" function to generate and download an accessible PDF file.

Another option to improve a Google Docs document is to export it as a Microsoft Word document (DOCX) and use Microsoft's accessibility tools.

Microsoft Word

After making the document accessible when writing it, run Microsoft Word's accessibility checker.

Currently, one of the quickest ways to check your document is to use the accessibility checker built into Microsoft Word. It works with any version of Word that was released after 2010. Here's how:

  1. In the main toolbar, select Review.
  2. Select Check accessibility.
  3. Review your results. You'll be presented with a list of errors, warnings, and tips with how-to-fix recommendations for each.
  4. Select a specific issue to see why and how you should fix it. To learn more, consult the list of rules for Microsoft's Accessibility Checker.
