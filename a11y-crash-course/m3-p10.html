How to make headings

Important: do not use the Title option in the headings menu. Both Google Docs and Microsoft Word have an option in the Heading menu referred to as Title. This can be very confusing when formatting a document that you want to have a title at the top.
The best way to make a document accessible is to have the main heading of the document — or a document title in most cases — be formatted as a Heading 1 (H1).

Microsoft Word

Organize your document with Word headings:

  1. Open your Word document for editing.
  2. Click or key your cursor into the text you want to present as a heading.
  3. In the Formatting toolbar (Home tab) click the Heading 1 button, OR use key combo ALT+O, S to open the style pane and
    key to the Heading 1 style option.
  4. Continue through the document, applying appropriately nested headings for each topic heading.
  5. Review the document outline in Navigation View (View→Navigation Pane).

Review Microsoft's Improve heading accessibility for the most up to date information.

Google Docs

Applying a heading style

  1. Select the text you want to apply a heading style to.
  2. Select the style menu, located between the zoom menu and the font menu.
  3. Select the appropriate heading level for your text.

Modifying your heading

You're using headings because it's the right thing to do — but you don't like how the pre-styled headings look. What's a content contributor to do?

  1. Write your heading text.
  2. Apply the correct heading style to it.
  3. Change the font, color, size, etc. until you are satisfied with your changes.
    Note: you can stop now if you want to change only this instance of the heading. If you'd like to give the same styling to all of your H2s, H3s, etc., continue to step 4.
  4. In the same headings menu as above, select the options arrow to the right of the heading level you are using.
  5. Select Update '[heading level]' to match.
