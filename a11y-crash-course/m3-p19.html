How to make tables

Microsoft Word

Watch this 2 minute video about creating accessible tables in Word.

Adding alt text to a table

Microsoft Word provides this option, though it is unclear whether adding alt text to tables increases their accessibility more than a well-crafted title, caption, or summary.

  1. Select Table Properties in the Table Layout tab.
    Note: You can also right-click on the table and select Table Properties from the menu that is displayed then.
  2. Select Alt Text.
  3. Enter a title and description for the table.

Adding header rows

  1. Choose Insert > Table to insert a table.
  2. Choose the number of rows and columns you want.
    Note: When you add a table to your document, two new tabs appear in the ribbon: Design and Layout. These are the Table Tools.
  3. On the Design tab, choose the Table Styles Options group, and then choose Header row. Other options include Banded Rows or Total Row.

Your table now has a header row. This means that, behind the scenes, Word and any assistive technologies can communicate intelligently about the table.

Another change you can make that will help sighted users is allowing header rows to be repeated at the top of each page. If your table happens to span multiple pages, the header row will appear at the top of each one; this makes the information in your table more accessible to many different types of visual learners. To repeat header rows:

  1. Place your cursor in the Header row.
  2. Select Table Properties.
  3. In the Row tab, check the box marked Repeat as header row across the top of each page.
  4. Select OK.

Adding column headings

  1. Place your cursor in the first cell on the top row of your new table.
  2. Type the name for this column and then press the tab key to move from one column to the next. Add additional column names as needed.

Your table now has column names, making it easier to understand the information that the table contains. Some screen readers can be set up to read column names at any time, which can help when working with a large table.

Google Docs

At present there is not a feature for adding alt text to a table in Google Docs. It may be helpful to add a 1-sentence caption or title summary as a header above the table so that anyone scanning through your document with a screen reader could quickly understand the table's purpose.

In Google Docs, the first row is automatically coded as a header row. This means that you should always make the first row of your table the header — do not add data in the first row.
