Skill 5: activity

With your apple practice document, implement the changes below.

  1. Set the "normal" or "paragraph" text style to at least 12pt size. Ensure the paragraph text is updated.
  2. Review the document's text for sections that would be easier to read as a list.
  3. Check that all text is left aligned and italics are used appropriately if at all.

How to modify text styles

Microsoft Word

Set the font type and size

The drop-down font and font size menus are located in the main editing toolbar, within the Home tab. Use font sizes of at least 12pt for print documents, and at least 14pt if you plan to share your screen in an online meeting.
Note: The font style and size are located above the bold/italic/underline options. While you are working in that part of the editing toolbar, un-italicize any italicized text.

Select a high contrast font color

The font color options are located in the main editing toolbar, within the Home tab. If you're unsure about the contrast, use Microsoft Word's built-in Accessibility Checker (covered later in this module) to see whether the font you are using has sufficient contrast with the background.

Align the text

Text alignment options are located to the right of the font color options in the main editing toolbar, within the Home tab. Align text to the left for left-to-right languages like English; align it to the right for right-to-left languages like Arabic.

Google Docs

Set the font

The font menu is located in the main editing toolbar, to the right of the Styles menu.

Set the font size

The font size menu is located to the right of the font menu. Use font sizes of at least 12pt for print documents, and at least 14pt if you plan to screenshare your document in an online meeting.

Set a high-contrast font color

The text color options are located next to the bold, italic, and underline options. While you are working in that part of the editing toolbar, be sure to un-italicize any italicized text.

Align the text

Text alignment options are located to the right of the image options in the main editing toolbar. They can also be reached by typing Command (Ctrl for PCs) + Shift + a letter — L for left, R for right, etc. Align text to the left for left-to-right languages like English; align it to the right for right-to-left languages like Arabic.
