Accessibility Crash Course

Skill 4: activity

With your document:

  1. For the apples added in the data table, link the common names to their own Wikipedia articles.
  2. Add a link to learn more about apples, linking it to the full Wikipedia article at https://en.wikipedia.org/wiki/Apple.
  3. Add a new bibliography section to the document, citing the sources for the content in the document so far. Read APA Style's Accessible URLs for guidance.

How to add links

Microsoft Word
  1. Select the text to which you want to add the hyperlink, and then right-click.
  2. Select Hyperlink. The text you selected displays in the Text to Display box. This is the text that the person will initially encounter.
  3. If necessary, change the hyperlink text. Make sure it is either the title of the destination or an accurate, descriptive summary of what the person will find there.
  4. In the Address box, type the destination URL. This is the actual website that the person will navigate to.
Google Docs
  1. In your document, highlight the text that you want to link.
  2. Select Insert and then Link, or type Command+K (ctrl+K).
  3. In the “Text” field, type the text you want to be linked. Make sure it is either the title of the destination or an accurate, descriptive summary of what the person will find there.
  4. In the “Link” field, enter a URL or email address or search for a website. You can also link to any headings that are already present in your document.
  5. Select Apply.
