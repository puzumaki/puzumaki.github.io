Document review with an accessibility checker

Accessibility checkers highlight areas in your document that might be problematic for people with disabilities, as well as for anyone using assistive technology. Accessibility checkers can:

  • catch accessibility mistakes,
  • provide guidance about how to fix accessibility errors, and
  • update according to changes in tools and accessibility-related laws.

Accessibility checkers cannot find all accessibility issues. In addition, they won't be able to tell you whether your content makes sense or is inaccurate. Accessibility checkers and critical thinking must work in tandem to create an accessible document.

How to check accessibility

Microsoft Word

Currently, one of the quickest ways to check your document is to use the accessibility checker built into Microsoft Word. It works with any version of Word that was released after 2010. This option makes it easy to go back and forth between Word and Google Docs. Here's how:

  1. In the main toolbar, select Review.
  2. Select Check accessibility.
  3. Review your results. You'll be presented with a list of errors, warnings, and tips with recommendations on how they can be fixed.
  4. Select a specific issue to see why and how you should fix it. To learn more, consult the list of rules for Microsoft's Accessibility Checker.
Google Docs

At present, Google Docs does not have a built-in accessibility checker. An accessibility-checking extension called Grackle Docs exists and is available through the University; it can be launched on a Google file to prepare it as an accessible PDF. Check the Office of Information Technology's article Use and Troubleshoot GrackleDocs Accessibility Checker article for more help using this tool.

Another way to check your document is to use the accessibility checker built into Microsoft Word:

  1. Open the File menu of your Google Doc.
  2. Select Download and then Microsoft Word (.docx); the file will be stored in your Downloads folder.
  3. Open the downloaded Microsoft Word file.
  4. In the main Microsoft Word toolbar, select Review.
  5. Select Check accessibility.
  6. Review your results. You'll be presented with a list of errors, warnings, and tips with how-to-fix recommendations for each.
  7. Select a specific issue to see why and how you should fix it. To learn more, consult the list of rules for Microsoft's Accessibility Checker.
  8. When you are finished, you can upload your file to Google Drive again. Accessibility changes made when the file was in Microsoft Word format are retained.
