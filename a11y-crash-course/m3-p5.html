Skill 1 activity

With your Google Doc or Microsoft Word activity document:

  1. Select an image from the apple Wikipedia page that is in the public domain. Download it.
  2. Add the image in the proper location of the document text. Then add the alternate text.

How to add alternate text and decorative elements

Microsoft Word

  1. Do one of the following:

    1. Select an image and press the Alt Text button in the Format ribbon tab.
      Note: Images have a Format Picture ribbon tab, charts have a Format Chart ribbon tab, etc.

    2. Right-click an image and select Edit Alt Text.
      The Alt Text pane opens on the right side of the document body.

  2. Describe the image content and context.

If the image is decorative and you intend to keep it, mark it as decorative:

  1. Right-click a visual element.

  2. Select Edit Alt Text. The Alt Text pane opens on the right side of the document body.

  3. Select the Mark as decorative check box. The text entry field will be grayed out.

Review Microsoft's Add alternative text to a shape, picture, chart, SmartArt graphic, or other object for the most up to date instructions.

Google Docs
  1. Select an image, drawing, or graphic.
  2. Right click and select Alt text.
    Note: You can also type Command+Option+Y on a Mac, or Ctrl+Alt+Y on a PC.
  3. Enter a title and description.
  4. Select OK.

Review the "include alt text" section of Google's Make your document or presentation more accessible.
