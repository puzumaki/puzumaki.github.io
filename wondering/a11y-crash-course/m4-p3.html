Limitations of PDF

When submitting a thesis or dissertation, you will likely be asked to save your document in the PDF format. If the original word document has been made using accessibility practices, making an accessible PDF can be straightforward.

PDFs are often misunderstood; however, they remain a primary file format in academic settings. Below are a few myths and limitations you may have heard.

Myth: PDF is secure and hard to change

This was true when the PDF file format was proprietary. However, the format was opened up in 2008 and was last updated by Adobe in 2011. It is no longer proprietary.

Free tools can easily remove PDF security features, and PDF editors are freely available (even Adobe has one). Microsoft Word can also import a PDF into an editable document.

Myth: PDF is best because other formats aren't universally compatible

Many formats are now open and work for all, including webpages (HTML) and Microsoft Word (DOCX), which became an open format in 2007 and can be edited in all word processing apps.

Additional limitations

PDFs have a few limitations that can make them difficult to use.

  • Accessibility was and still is an afterthought. Improving accessibility of a PDF continues to be challenging.
  • Reading a PDF on a mobile screen is often difficult. Some apps can adapt if the PDF is accessible.
  • If you have an inaccessible PDF, remediating it to be accessible is difficult, time consuming, and expensive. Structure tags and reading order are especially complicated and finicky.
