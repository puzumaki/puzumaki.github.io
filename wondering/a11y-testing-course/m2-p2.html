Accessibility Testing Course

Step 1: submit a page for evaluation

The Web Accessibility Visual Evaluator, WAVE, is a free tool administered by WebAIM that helps locate and identify accessibility problems on web pages so they can be repaired. This is the tool we'll use for the course.

It analyzes one page at a time for errors, possible errors called alerts, semantic elements, and structural elements. WAVE requires a basic understanding of accessibility, but it also helps you learn.

Pages can be evaluated using two methods:

  • putting an URL into the WAVE webpage, or
  • using an installed browser extension for Chrome, Firefox, or Microsoft Edge.

Method 1: the WAVE web page

  1. Open a web browser
  2. Go the the WAVE home page at https://wave.webaim.org
  3. Enter the address of a web page that is publicly accessible on the Internet in the text box.
  4. Hit your enter or return button to submit the request for page analysis.

After a few seconds, the report should appear in your browser.

Method 2: a browser extension

  1. Open a supported browser, either Chrome, Firefox, or Microsoft Edge.
  2. Go to the WAVE extensions page at https://wave.webaim.org/extension/
  3. Use the link to install the browser extension from the list of browsers.
  4. Once installed, navigate to the web page you want to submit for page analysis.
  5. Open the WAVE extension in the browser.

The report will load as an overlay on that web page.
